# You G?

A social scheduling app that helps groups of friends find the best time to meet — share availability, auto-detect overlapping free time, create events, vote on times and locations, and get reminded.

Built as a production-quality portfolio project following real software engineering practices: Clean Architecture, Repository Pattern, SOLID, feature-first structure, tested and CI/CD-driven from day one.

## Stack

- **Frontend**: Flutter, Riverpod, GoRouter, Dio, Material 3
- **Backend**: ASP.NET Core Web API (.NET 9), EF Core, PostgreSQL, JWT auth, MediatR (CQRS-lite)
- **Infrastructure**: Docker, Docker Compose, GitHub Actions, Railway
- **Notifications**: Firebase Cloud Messaging
